## Translation Guidelines / 翻译指南

### General Principles / 一般原则

1. **Consistency / 一致性**: Always use the same Chinese translation for a given English term throughout all documentation.
   始终在所有文档中对给定的英文术语使用相同的中文翻译。

2. **Clarity / 清晰度**: Choose translations that are clear and commonly understood in the Chinese bioinformatics and machine learning communities.
   选择在中国生物信息学和机器学习社区中清晰且常用的翻译。

3. **Keep Technical Terms / 保留技术术语**: Some technical terms (like model names, file formats) should be kept in English.
   某些技术术语（如模型名称、文件格式）应保留英文。

4. **Context Matters / 上下文很重要**: Some words may have different translations depending on context. Use professional judgment.
   有些词汇根据上下文可能有不同的翻译。请使用专业判断。

### Special Cases / 特殊情况

- **Model Names**: Keep model names in English (e.g., DNABERT, EVO, Mamba)
  **模型名称**：保留英文模型名称（例如 DNABERT、EVO、Mamba）

- **Code Elements**: Code snippets, function names, and variable names remain in English
  **代码元素**：代码片段、函数名和变量名保持英文

- **Abbreviations**: Common abbreviations (GPU, CPU, API) are kept in English
  **缩写**：常见缩写（GPU、CPU、API）保持英文

- **File Extensions**: File extensions (.py, .yaml, .json) remain in English
  **文件扩展名**：文件扩展名（.py、.yaml、.json）保持英文
